Release v0.8.1 - Critical Network Stability Fixes

This patch release resolves critical concurrency issues, memory leaks, and connection problems in the networking layer.

Key Fixes:
- Application startup hanging resolved
- Memory leaks eliminated (CancellationTokenSource, NetworkStream)
- Race conditions fixed in all network services
- Connection timeouts and keepalive implemented
- Automatic reconnection on failures
- Max concurrent connections limit (DoS protection)
- Thread-safe connection management
- Enhanced error handling and logging

Production-ready for .NET MAUI and Avalonia distributed applications.

Full changelog in commit 3b621d3.
